# Sales Data Analysis Project

## Overview
This project analyzes a sales dataset using Python and the Pandas library to extract meaningful business insights.

## Dataset Summary
- Total Records: 100
- Total Columns: 7
- No missing or duplicate values were found.

## Tools Used
- Python
- Pandas

## Analysis Performed
- Loaded and explored the sales dataset
- Cleaned and validated the data
- Calculated key sales metrics
- Identified the best-selling product

## Key Results
- Total Revenue: ₹12,365,048
- Average Sale Value: ₹123,650.48
- Highest Sale: ₹373,932
- Lowest Sale: ₹6,540
- Best-Selling Product: Laptop

## Conclusion
The analysis shows that Laptop is the top-performing product in terms of total sales. The dataset was clean and suitable for further analysis such as visualization or trend analysis.
